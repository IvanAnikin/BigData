# BigData

## Code
matrix.py           - intuitive approach

Matrix.java         - intuitive approach

## Requirements
- Python (3.x)
- Java (JDK)
- C (gcc/g++ compiler)
+ libraries

## Usage (Windows)

Python:
```
pip install requirements.txt
python matrix.py
```

C:
```
gcc -o matrix_c.exe matrix.c -lpsapi
matrix_c.exe
```

Java:
```
javac Matrix.java
java Matrix
```
